# Diff Checker

Side-by-side text diff with line-level change highlighting — paste two texts and see exactly what changed.

## Features

- Side-by-side and unified diff views
- Line-level highlighting: additions in green, deletions in red
- Character-level diff within changed lines
- Line numbers
- Copy output button
- Supports code, JSON, plain text — anything text-based
