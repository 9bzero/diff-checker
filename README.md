# diff-checker

Side-by-side text diff. Paste two blocks of text and see what changed — added, removed, unchanged.

## Features

- Line-by-line diff with color highlighting
- Added lines (green), removed lines (red), unchanged (gray)
- Character-level diff within changed lines
- Line numbers
- Unified diff view toggle
- Copy diff as patch

## Run

```bash
npm install && npm run dev
```
